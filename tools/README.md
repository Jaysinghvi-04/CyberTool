# Python Keylogger (Educational)

This is a basic keylogger built with Python and the `pynput` library. It records keystrokes to a local file.

**Usage:**
1. Install dependencies: `pip install -r requirements.txt`
2. Run the tool: `python src/main.py`
3. Press `Esc` to stop the recording.
